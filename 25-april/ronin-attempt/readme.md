## Setup

Need this `.env`:

```
# -- db
RONIN_TOKEN=
JWT_SECRET=

# -- email
RESEND_API_KEY=
```

- `RONIN_TOKEN` - get from [ronin.co](https://ronin.co)
- `JWT_SECRET` - can enter any string
- `RESEND_API_KEY` - can get from [resend.com](https://resend.com) (not needed in dev)

```
bun i
```

## Run

```
bun dev
```
